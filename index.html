Concepts and Definitions

Term Definition Image
Corrient The flow of electric charge (measured in Amperes).
Ampere The unit of electric current (A) .
Resistor A component that resists the flow of current.
Voltage The electric potential difference (measured in Volts).
Ohm’s Law V=I×R (Voltage = Current × Resistance).
LDR A resistor whose resistance changes with light intensity.
LED A diode that emits light when current flows through it.
DC Motor A motor that runs on direct current (DC).
Motor Reducer A gear system that reduces motor speed while increasing torque.
Servo Motor A motor with precise angular control (used in robotics).
Step Motor A motor that moves in discrete steps (used for precise positioning).
H-Bridge A circuit that controls the direction of a DC motor.
Transistor A semiconductor device used for switching or amplification.
Integrated Circuit A miniaturized electronic circuit on a chip.
Potentiometer A variable resistor used for adjusting voltage.
Switch A device that opens or closes a circuit.
Wire A conductor for electric current.
Battery A device that stores and supplies electrical energy.
Buzzer A component that produces sound when voltage is applied.
Microphone Converts sound waves into electrical signals.
Sensor A device that detects physical input (e.g., light, heat).
Thermistor A resistor whose resistance changes with temperature.
Flame Sensor Module Detects fire or infrared light from flames.
Light Sensor Module Measures ambient light intensity.
Smoke Sensor Module Detects smoke particles in the air.
Vibration Sensor Module Detects vibrations or shocks.
Tilt Sensor Module Detects orientation changes.
Noise Sensor Module Measures sound levels.
Touch Sensor Module Detects physical touch or proximity.
Infrared Obstacle Avoidance Module Uses IR to detect obstacles.
Infrared Line Tracking Module Follows lines using IR reflection.
PS2 Joystick Module A two-axis analog joystick for input control.
Microcontroller A small computer on a single chip (e.g., Arduino, ESP32).
Programming Language A set of instructions for computers (e.g., Python, C++).
Python A high-level, interpreted programming language.
MicroPython A Python implementation for microcontrollers.
C++ A general-purpose programming language with object-oriented features.
IDE Software for coding, compiling, and debugging.
Arduino An open-source electronics platform.
Arduino UNO A popular Arduino board based on the ATmega328P.
Arduino IDE The official software for programming Arduino boards.
ESP32 A low-cost, Wi-Fi & Bluetooth-enabled microcontroller.
